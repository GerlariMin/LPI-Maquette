Première chose récupérer le dossier BarberLife-Project avec la commande git pull sur le git Master

- Pour récupérer toutes les dépendances que j'ai, faire la commande npm Install dans le dossier BarberLife-Project

- Puis ouvrez le dossier DB et le fichier route.js une fois la base de données créer vous devez mettre les info de la base dans la variable connection.

- Une fois fait votre connexion devrais marcher pour vérifier aller dans le terminal dans le dossier DB et faite Node route.js si vous voyez cool s'afficher c'est
que la connection est établie

Au cas ou :

- Si la connection ne marche pas installer les dépendances vous même avec dans le dossier BarberLife-Project :
    * npm install express
    * npm install body-parser
    * npm install mysql
